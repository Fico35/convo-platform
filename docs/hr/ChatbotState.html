ChatbotState

Predstavlja stanje u kojem konverzacijski agent može biti. Ima vlastite akcije, odgovore i prijedloge, odvojeno od drugih stanja.

Funkcije za dodavanje akcija, odgovora i prijedloga su dostupne, ali za uklanjanje istih nisu. Tako je postavljeno zato što se stanje ne bi trebalo mijenjati nakon što se postavi. Ako je potrebno promijeniti ponašanje konverzacijskog agenta, stvorite novo stanje s prikladnim akcijama, odgovorima i prijedlozima te promijenite trenutno stanje konverzacijskog agenta u novo stanje.

Konstruktor

new ChatbotState(name);

Parametar Tip Opis
name string String koji predstavlja ime stanja. Mora biti jedinstveno (za ovu instancu konverzacijskog agenta).

Svojstva

  • name
  • actions
  • answers

Metode

  • addAction
  • addAnswer
  • process

Svojstva

.name

String koji je predani koa ime kroz konstruktor. Ovo ime se koristi za ključ u mapi svih stanja u instanci konverzacijskog agenta (Chatbot).
Tip: string

.actions

Mapa koja sadrži akcije dodijeljene namjerama. Ključ je namjera, vrijednost je akcija. Kada se metoda process pozove s namjerom koja postoji u ovoj mapi, odgovarajuća akcija se poziva s ovim stanjem kao parametrom.
Tip: Map<string, function>

.answers

Mapa koja sadrži odgovore dodijeljene namjerama. Ključ je namjera, vrijednost je polje odgovora (više odgovora može biti dodijeljeno istoj namjeri). Kada se metoda process pozove s namjerom iz ove mape, svi odgovarajući odgovori (cijelo polje) se vraćaju kao dio rezultata. Napomena: ako se koristi funkcionalnost uvoza/izvoza, odgovori bi trebali biti stringovi (jer se kod uvoza i izvoza tako koriste).
Tip: Map<string, Array<any>>

Metode

.addAction( intent, action )

Koristi se kod dodavanja akcije koja će se izvršiti kada se prepozna zadana namjera. Dozvoljena je samo jedna akcija po namjeri, pa će zato ponovno pozivanje ove funkcije s istim intent parametrom (istom namjerom) rezultirati s prepisivanjem nove akcije preko stare. Vraća true ako nije postojala akcija za zadanu namjeru prije, inače vraća false.
Parametar Tip Opis
intent string Namjera koju treba prepoznati
action function Akcija koja će se izvršiti

Povratna vrijednost: boolean

.addAnswer( intent, answer )

Koristi se za dodavanje odgovora koji se trebaju vratiti za zadanu namjeru. Ako je zadani odgovor polje, svi elementi polja se dodaju umjesto dodavanja samog polja kao odgovora. Vraća true ako prije nije bilo odgovora za zadanu namjeru, inače vraća false.
Parametar Tip Opis
intent string Namjera koju treba prepoznati
answer any or Array<any> Odgovor (ili odgovori) koji se vraća za zadanu namjeru

Povratna vrijednost: boolean

.process( intent )

Obrađuje namjeru. Poziva ju konverzacijski agent (Chatbot) kada je ovo stanje trenutno stanje agenta. Izvršava odgovarajuću akciju za namjeru (ako postoji) i vraća namjeru s zadanim odgovorima i prijedlozima (ako su definirani).
Parametar Tip Opis
intent string Namjera za obraditi

Povratna vrijednost: Promise<any[]>
