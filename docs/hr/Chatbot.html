Chatbot

Predstavlja instancu konverzacijskog agenta koji se može koristiti za obradu poruka.

Konstruktor

new Chatbot(intentExtractionFunction);

Parametar Tip Opis
intentExtractionFunction function Funkcija koju ovaj konverzacijski agent koristi kako bi odredio namjeru iz rečenice

Svojstva

  • extractIntent
  • allStates
  • currentState

Metode

  • addState
  • getState
  • setCurrentState
  • importStates
  • importStatesWithActions
  • exportStates
  • exportStatesWithActions
  • process

Svojstva

.extractIntent

Funkcija koja je konverzacijskom agentu predana kroz konstruktor. Ova instanca konverzacijskog agenta koristi ovu funkciju kod određivanja namjere iz rečenice. Funkcija prima rečenicu (string) kao parametar, a vraća objekt sa sljedećim svojstvima: 'utterance', 'intent', 'score'.
Tip: function

.allStates

Mapa koja sadrži sva stanja ove instance konverzacijskog agenta. Koristi se kod mijenjanja trenutnog stanja.
Tip: Map<string, ChatbotState>

.currentState

Trenutno aktivno stanje. Koristi se kod obrade namjere poruke (dobivanje odgovora i prijedloga).
Tip: ChatbotState

Metode

.addState( state )

Koristi se za dodavanje objekta tipa ChatbotState u mapu svih stanja. Vraća true ako je stanje uspješno dodano, inače baca grešku.
Parametar Tip Opis
state ChatbotState Stanje koje se dodaje u konverzacijski agent

Povratna vrijednost: boolean

.getState( stateName )

Koristi se za dohvaćanje stanja iz mape svih stanja koristeći string koji predstavlja ime stanja.
Parametar Tip Opis
stateName string Ime stanja za dohvatiti

Povratna vrijednost: ChatbotState

.setCurrentState( stateName )

Koristi se za postavljanje trenutnog stanja konverzacijskog agenta. Koristite ovo umjesto direktnog postavljanja svojstva currentState jer ova metoda provjerava ako stanje postoji. Vraća true ako je stanje uspješno promijenjeno, inače baca grešku.
Parametar Tip Opis
stateName string Ime stanja koje se postavlja na trenutno

Povratna vrijednost: boolean

.importStates( stateJson )

Uvozi stanja agenta s odgovorima i prijedlozima, bez akcija. JSON koji se koristi bi trebao biti rezultat iz prije pozvane metode exportStates ili exportStatesWithActions. Funkcija vraća Promise koji se razrješava prazno (undefined).
Parametar Tip Opis
stateJson JSON JSON string koji sadrži stanja s odgovorima i prijedlozima

Povratna vrijednost: Promise<undefined>

.importStatesWithActions( stateJson )

Uvozi stanja sa svime: akcijama, odgovorima i prijedlozima. Pošto su akcije tekst koji će se pretvoriti u izvršni kod, ovu metodu treba oprezno koristiti. Samo koristite ovo ako je JSON Vaš vlastiti ili dolazi iz sigurnog izvora. JSON bi trebao biti rezultat prije pozvane exportStates ili exportStatesWithActions metode. Funkcija vraća Promise koji se razrješava prazno (undefined).
Parametar Tip Opis
stateJson JSON JSON string koji sadrži stanja s akcijama, odgovorima i prijedlozima

Povratna vrijednost: Promise<undefined>

.exportStates( )

Izvozi sva stanja konverzacijskog agenta u JSON obliku. Izvezena stanja imaju samo odgovore i prijedloge bez akcija. Funkcija vraća Promise koji se razrješava s rezultantnim JSON objektom.
Povratna vrijednost: Promise<JSON>

.exportStatesWithActions( )

Izvozi sva stanja konverzacijskog agenta u JSON obliku. Izvezena stanja sadrže sve: akcije, odgovore i prijedloge. Pošto su akcije izvršni kod dobiven iz teksta, pri uvozu treba biti oprezan. Funkcija vraća Promise koji se razrješava s rezultantnim JSON objektom.
Povratna vrijednost: Promise<JSON>

.process( utterance )

Obrađuje korisnikovu poruku. Najprije se poziva extractIntent metoda kako bi se dobila namjera iz parametra utterance. Zatim se namjera koristi kao parametar pri pozivu process metode trenutnog stanja. Funkcija vraća Promise koji se razrješava s rezultantnim objektom.
Parametar Tip Opis
utterance string Poruka od korisnika koju treba obraditi

Povratna vrijednost: Promise<{utterance, intent, score, answers}>
